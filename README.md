# ❄️ RefrigSim Demo v2.0

### Simulador Educacional do Ciclo de Refrigeração

O **RefrigSim Demo** é um software desenvolvido em Python para auxiliar estudantes, técnicos, professores e engenheiros na análise do ciclo de compressão de vapor.

Utilizando propriedades termodinâmicas reais através da biblioteca **CoolProp**, o programa permite simular diferentes condições de operação e compreender o comportamento dos principais parâmetros utilizados na refrigeração.

## 📘 Manual Técnico

O RefrigSim Demo acompanha um **Manual Técnico** desenvolvido para auxiliar estudantes, técnicos e engenheiros na compreensão dos principais parâmetros utilizados nas simulações.

O manual aborda:

- Temperatura de evaporação;
- Temperatura de condensação;
- Superaquecimento;
- Sub-resfriamento;
- Eficiência do compressor;
- Carga térmica;
- Perdas de pressão na sucção;
- Perdas de pressão na descarga.
